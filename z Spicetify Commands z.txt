
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

SPICETIFY COMMANDS
\\POWERSHELL\\ (no admin needed)

~~~~~~~~~~~~~~~~~

INSTALL SPICETIFY:
iwr -useb https://raw.githubusercontent.com/spicetify/spicetify-cli/master/install.ps1 | iex

(Install Marketplace by using `y` when prompted)

Then run:

spicetify backup apply

~~~~~~~~~~~~~~~~~

UNINSTALL SPICETIFY:
spicetify restore backup apply

Remove-Item "$env:APPDATA\spicetify" -Recurse -Force
Remove-Item "$env:LOCALAPPDATA\spicetify" -Recurse -Force

IF INSTALLED BY SCOOP:
scoop uninstall spicetify

TEST IF ITS GONE:
spicetify

~~~~~~~~~~~~~~~~~

APPLY THEME
spicetify config current_theme <nameofthemefolder>          
spicetify apply

~~~~~~~~~~~~~~~~~

IF THEME COMES WITH MULTIPLE COLOR SCHEMES:
spicetify config current_theme <nameofthemefolder>  
spicetify config color_scheme <nameofcolorscheme>   
spicetify apply

~~~~~~~~~~~~~~~~~

UNAPPLY A THEME
spicetify config current_theme SpicetifyDefault
spicetify apply

~~~~~~~~~~~~~~~~~

CHECK APPLIED THEME
spicetify config current_theme

~~~~~~~~~~~~~~~~~

APPLY EXTENSIONS
spicetify config extensions <nameofextension.js> 

**(if your file is .mjs and doesnt work after applying then rename it to .js)**

~~~~~~~~~~~~~~~~~

UNAPPLY EXTENSION
spicetify config extensions <nameofextension.js>-         <---- Minus sign at the end

~~~~~~~~~~~~~~~~~

CHECK APPLIED EXTENSIONS
spicetify config extensions

~~~~~~~~~~~~~~~~~

APPLY CUSTOM APPS:
spicetify config custom_apps <nameofcustomappfolder>

~~~~~~~~~~~~~~~~~

UNAPPLY CUSTOM APPS:
spicetify config custom_apps <nameofcustomappfolder>-     <---- Minus sign at the end

~~~~~~~~~~~~~~~~~

CHECK APPLIED CUSTOM APPS
spicetify config custom_apps

~~~~~~~~~~~~~~~~~

VIEW YOUR SPICETIFY CONFIGURMENT:
spicetify config

~~~~~~~~~~~~~~~~~

SPICETIFY RESET:
spicetify config current_theme SpicetifyDefault
spicetify restore
spicetify backup
spicetify apply

~~~~~~~~~~~~~~~~~

OTHERS:
spicetify enable-devtools
